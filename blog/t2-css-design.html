  • padding: I understand this as the distance between the context and the border.
  • border: I understand this as the distance between the edge of the padding and the margin.
  • Margin: I understand this as the very edge of a box. And a margin can be thickened to be seen as a frame.

    • I feel this image give a very good visual description of what each of these means. It's a bit tricky to try and explain how to use these in conjuction with positioning because I don't know how to describe positioning quite yet. I will try to explain how to use padding, border, and margin alone. Let's say you've uploaded an image and you want to use these. The padding will give you space between the image and the border. kind of like the matte board inside a house frame holding a photo. So you can set it to be as thin or thick as you want; however, unlike the matte board, if the padding is set too small, the image will be shrunk down in attempt to fit. Now the border, Think of this like the frame of that photo. You can have as big a frame as you want but not smaller than the padding. The Margin, is like the wall on which you'd hang that frame. That all depends on you and where you want the image to be on that wall.
